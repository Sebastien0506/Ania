# Ania

### Ania est un bot discord développé en Python. Il utilise le framework Django pour la parti backend et Angular pour la parti frontend.

### Ania peut rejoindre le salon vocal d'un utilisateur, lire les messages des différents salons textuels du serveur, parler dans un salon vocal et gérer certains paramètres du serveur, selon les autorisation accordées par l'administrateur.
